
# Plasma Setup

Setup Script for KDE Plasma on Alpine Linux

## Features
Run the script and you will see.

## Requirements
- Bash
- Git
- Network Connection

## Installation
```doas apk add bash git```

```git clone https://github.com/Jagermist/alpine-setup.git --depth=1```
## Usage
`cd alpine-setup`

`bash ./main.sh`

Note: the script is 100% interactive
